# Awakening by Ksenia - התעוררות על ידי קסניה

## 🎯 **סקירה כללית**
**Awakening by Ksenia** הוא **מרחב אפליקטיבי להרחבת התודעה** – פלטפורמת ווב/PWA מתקדמת שתכלול מגוון שירותים וכלים רוחניים ומדעיים-אינטואיטיביים.

בגרסה הראשונה זמינה התוספת **"מחשבון קוד הכסף"** (Numerology Money Code). בהמשך יתווספו:
- **מצפן Feng Shui** (כיוונים אישיים וחלוקת בית/עסק)
- **נומרולוגיה ואנרגיות היום**
- **Human Design בסיסי ומתקדם**
- **כלים משלימים** (מדיטציות, יומני תרגול, ועוד)

**שורת תיוג**: **"Awakening by Ksenia — מרחב לתודעה מתקדמת. התחל/י עם מחשבון קוד הכסף."**

## 🚀 **טכנולוגיות מרכזיות**
- **Next.js 14** עם App Router
- **React 18** עם TypeScript
- **Tailwind CSS 3** + עיצוב מותאם אישית
- **Vercel** לפריסה ואחסון
- **PWA** (Progressive Web App) עם יכולות offline

## 📁 **מבנה הפרויקט**
```
├── app/                    # עמודי האפליקציה (App Router)
│   ├── layout.tsx         # פריסה ראשית
│   ├── page.tsx           # עמוד הבית
│   ├── privacy/page.tsx   # מדיניות פרטיות
│   └── numbersmeaning/page.tsx # פירושי המספרים
├── public/                # קבצים סטטיים
│   ├── logo.svg          # לוגו ראשי
│   ├── icon.svg          # אייקון עם רקע למסך הבית
│   ├── iconvector.svg    # אייקון ללא רקע
│   ├── signature.svg     # חתימה
│   ├── numbersmeaning.html # תוכן פירושי המספרים
│   └── privacytext.html  # תוכן מדיניות הפרטיות
├── styles/               # סגנונות
│   └── globals.css       # סגנונות גלובליים
└── components/           # רכיבים נוספים (לעתיד)
```

## 🎨 **מערכת עיצוב**

### **צבעים (CSS Variables)**
```css
--gold: #D4B896        /* זהב */
--gold-deep: #C6A170   /* זהב עמוק */
--ivory: #FEFEFE       /* שנהב */
--charcoal: #1F2024    /* פחם */
--text-secondary: #6B6F76 /* טקסט משני אפור */
```

### **פונטים**
- **Assistant בלבד** - פונט נקי וקריא לעברית ואנגלית
- טעינה דרך Google Fonts
- משקלים: 300, 400, 500, 600, 700, 800

### **טיפוגרפיה והדגשות**
- **כותרות H1–H3** בצבע זהב עמוק ומרכוזות
- **הדגשות** (`<strong>`) בצבע פחם/שחור
- **יישור למרכז** כברירת מחדל (מובייל ודסקטופ)
- **רספונסיבי** - גדלי פונטים מותאמים למסכים שונים

## 🔧 **התקנה והפעלה**

### **סביבת פיתוח**
```bash
# התקנת התלותים
npm install

# הרצת שרת פיתוח
npm run dev

# בניה לפרודקציה
npm run build

# הרצת שרת פרודקציה
npm start
```

### **פריסה ל-Vercel**
```bash
# התחברות לחשבון Vercel
vercel login

# פריסה ראשונה
vercel

# פריסות עתידיות
vercel --prod
```
