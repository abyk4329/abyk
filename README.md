# Awakening by Ksenia - התעוררות על ידי קסניה

## 🎯 **סקירה כללית**
**Awakening by Ksenia** הוא **מרחב אפליקטיבי להרחבת התודעה** – פלטפורמת ווב/PWA מתקדמת שתכלול מגוון שירותים וכלים רוחניים ומדעיים-אינטואיטיביים.

בגרסה הראשונה זמינה התוספת **"מחשבון קוד הכסף"** (Numerology Money Code). בהמשך יתווספו:
- **מצפן Feng Shui** (כיוונים אישיים וחלוקת בית/עסק)
- **נומרולוגיה ואנרגיות היום**
- **Human Design בסיסי ומתקדם**
- **כלים משלימים** (מדיטציות, יומני תרגול, ועוד)

**שורת תיוג**: **"Awakening by Ksenia — מרחב לתודעה מתקדמת. התחל/י עם מחשבון קוד הכסף."**

## 🚀 **טכנולוגיות מרכזיות**
- **Next.js 14** עם App Router
- **React 18** עם TypeScript
- **Tailwind CSS 3** + עיצוב מותאם אישית
- **Vercel** לפריסה ואחסון
- **PWA** (Progressive Web App) עם יכולות offline

## �️ **הגדרה מהירה למערכת התשלומים**

### 1. הגדרת משתני סביבה
ערכי את הקובץ `.env.local`:
```env
EMAIL_USER=your-email@gmail.com
EMAIL_PASSWORD=your-gmail-app-password
WEBHOOK_SECRET=your-secret-key
```

### 2. הגדרת Gmail App Password
1. עבור ל-[Google Account Security](https://myaccount.google.com/security)
2. הפעילי "2-Step Verification"
3. צרי "App Password" חדש
4. השתמשי בסיסמה זו ב-`EMAIL_PASSWORD`

### 3. הגדרת Webhook במערכת התשלומים
- **Webhook URL**: `https://your-domain.vercel.app/api/payment-webhook`
- **Method**: POST
- **תשלח לקוח פירוש**: לאחר תשלום מוצלח ✅

### 4. פריסה לפרודקציה
```bash
vercel --prod
```
